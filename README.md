# Phone Book API

## Prerequisites

- .Net Core SDK
- MySQL Server

## Setup

1. Install .Net Core 2.2 SDK (https://dotnet.microsoft.com/download) 
2. Install MySQL Server (https://dev.mysql.com/downloads/installer/)
3. Run the SQL Script on MySQL Server
    - You can find it in the root folder, called `PhoneBookScript.sql`

## Running API
- In the terminal, run command: `dotnet run -p phone-book-api/phone-book-api.csproj`
- Or alternatively, open the project in Visual Studio and Run the project `phone-book-api`
